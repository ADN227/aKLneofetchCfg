# Requisitos

- [neofetch](https://github.com/dylanaraps/neofetch)

        pacman -S neofetch

- [nerd fonts](https://github.com/ryanoasis/nerd-fonts)

        yay -S nerd-fonts-hack

- [lolcat](https://github.com/busyloop/lolcat)

        pacman -S lolcat
